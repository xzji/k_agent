# 支付尾款并开具发票指南

## 一、任务概述

按照合同约定支付剩余车款（全款或放款），并要求经销商开具机动车销售统一发票。

---

## 二、支付尾款

### 2.1 支付方式

| 支付方式 | 适用情况 | 操作要点 |
|---------|---------|---------|
| **全款支付** | 一次性付清全款 | • 确认尾款金额<br>• 通过银行转账/刷卡/现金支付<br>• 保留付款凭证 |
| **贷款放款** | 选择按揭/贷款购车 | • 等待银行放款<br>• 确认放款到账<br>• 与经销商核对到账金额 |

### 2.2 尾款支付流程

```
确认尾款金额 → 选择支付方式 → 完成支付 → 获取付款凭证 → 与经销商确认收款
```

### 2.3 支付前确认事项

- [ ] 核对合同约定的尾款金额
- [ ] 确认车辆信息无误（车架号、发动机号等）
- [ ] 检查车辆外观、内饰及各项功能
- [ ] 确认赠品和加装项目已兑现
- [ ] 了解发票开具所需时间

### 2.4 付款凭证保存

付款后务必保留以下凭证：
- 银行转账回单
- 刷卡小票/POS单
- 收据
- 电子转账截图

---

## 三、开具发票

### 3.1 发票类型

**机动车销售统一发票**

- 是办理车辆上牌、缴纳购置税、保险等手续的必备凭证
- 一车一票，具有唯一性
- 发票金额为实际购车价格

### 3.2 发票开具所需信息

### 个人购车

| 信息项 | 说明 |
|-------|------|
| 购买方名称 | 个人姓名 |
| 身份证号码 | 与合同一致 |
| 联系电话 | 便于后续联系 |
| 购买方地址 | 常住地址 |

### 单位购车

| 信息项 | 说明 |
|-------|------|
| 购买方名称 | 单位全称 |
| 纳税人识别号 | 统一社会信用代码 |
| 地址电话 | 单位注册地址及电话 |
| 开户行及账号 | 用于付款结算 |

### 3.3 发票开具流程

```
提供开票信息 → 经销商填写开票系统 → 税务系统验证 → 打印发票 → 交付购车人
```

### 3.4 发票核对要点

收到发票后，请仔细核对：

- [ ] **车辆信息**：品牌型号、车辆识别代号（车架号）、发动机号
- [ ] **购买方信息**：名称、身份证号/统一社会信用代码准确无误
- [ ] **金额信息**：价税合计金额与实际付款金额一致
- [ ] **日期信息**：开票日期正确
- [ ] **经销商信息**：销售方名称、纳税人识别号、地址电话
- [ ] **发票专用章**：清晰完整，无遮挡

### 3.5 发票注意事项

⚠️ **重要提醒**

1. **发票不得涂改**：任何信息错误需退回重开
2. **发票金额影响税费**：购置税按发票不含税价计算
3. **发票遗失麻烦**：务必妥善保管原件
4. **电子发票**：部分地区支持电子发票，同样具有法律效力

## 四、常见问题

### Q1：尾款可以晚几天支付吗？

> A：建议按合同约定时间支付，避免产生违约金或影响提车。如需延期，提前与经销商协商。

### Q2：发票金额可以开低一些吗？

> A：不建议。机动车销售统一发票必须按实际交易金额开具，低开发票涉嫌逃税，且影响车辆保值。

### Q3：发票开错了怎么办？

> A：发现错误立即联系经销商，在未上牌前可以退回重开。上牌后无法重开发票。

### Q4：全款和贷款付款时间有区别吗？

> A：有。全款支付通常当天可完成；贷款需等待银行审批和放款，一般3-7个工作日。

### Q5：单位购车需要额外提供什么材料？

> A：除基本开票信息外，可能需要提供营业执照复印件、授权委托书、经办人身份证等。

- [ ] ✅ 车辆使用说明书

## 六、下一步操作

完成尾款支付和发票开具后，即可进入下一阶段：

1. **购买车辆保险**（交强险、商业险）
2. **缴纳车辆购置税**
3. **办理车辆上牌**
4. **领取正式牌照和行驶证**

## 七、总结与关键信息

### 主要发现

1. **尾款支付**是购车流程中资金结算的最后环节，需严格按合同执行
2. **机动车销售统一发票**是车辆合法身份的重要凭证，影响后续所有手续办理
3. **发票信息核对**至关重要，任何错误都会带来麻烦

### 关键信息

| 项目 | 关键点 |
|-----|-------|
| 尾款金额 | 以合同约定为准，支付前二次确认 |
| 支付凭证 | 务必保存原件，作为法律凭证 |
| 发票类型 | 机动车销售统一发票（一车一票） |
| 开票信息 | 个人需身份证，单位需税号等信息 |
| 发票核对 | 车架号、发动机号、金额、购买方信息 |
| 发票保管 | 原件上交车管所，提前拍照备份 |

### 风险提示

- ⚠️ 付款前务必验车，确认无误后再支付
- ⚠️ 发票信息错误会导致上牌失败，必须当场核对
- ⚠️ 保留所有付款凭证，以备纠纷时维权使用
- ⚠️ 谨防经销商要求开具更低金额发票
