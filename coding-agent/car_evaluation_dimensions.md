# 车型对比评估维度体系

## 一、评估维度总览

| 维度类别 | 权重建议 | 评估项数量 |
|---------|---------|-----------|
| 经济性维度 | 25% | 4项 |
| 性能维度 | 20% | 4项 |
| 空间与舒适性 | 20% | 4项 |
| 配置与科技 | 15% | 3项 |
| 品牌与售后 | 10% | 3项 |
| 安全性 | 10% | 3项 |

---

## 二、详细评估维度

### 1. 经济性维度（权重：25%）

#### 1.1 价格竞争力（权重：8%）
| 评估指标 | 评分标准（满分10分） |
|---------|---------------------|
| 裸车价格 | 与同级竞品对比，价格越低得分越高 |
| 优惠力度 | 官方优惠+终端优惠综合评估 |
| 金融政策 | 贷款利率、首付比例、免息期等 |
| 购置税政策 | 是否享受购置税减免或补贴 |
| **总分计算** | 10分制 |

#### 1.2 油耗/续航表现（权重：8%）
| 评估指标 | 评分标准 |
|---------|---------|
| **燃油车** | |
| 官方油耗（L/100km） | 6L以下=10分，6-7L=8分，7-8L=6分，8-9L=4分，9L以上=2分 |
| 实际油耗反馈 | 参考车主实际油耗数据 |
| 油品要求 | 92号汽油=10分，95号=8分，98号=5分 |
| **新能源车** | |
| CLTC续航里程 | 600km以上=10分，500-600km=8分，400-500km=6分，300-400km=4分，300km以下=2分 |
| 实际续航达成率 | 80%以上=10分，70-80%=8分，60-70%=6分，50-60%=4分，50%以下=2分 |
| 充电效率 | 支持快充且功率>100kW=10分，50-100kW=7分，<50kW=4分 |

#### 1.3 保养维护成本（权重：5%）
| 评估指标 | 评分标准 |
|---------|---------|
| 首保费用 | 免费=10分，<500元=8分，500-1000元=5分，>1000元=2分 |
| 常规保养周期 | 1万公里/6个月以上=10分，5000公里/3个月=5分 |
| 年均保养成本 | <1000元=10分，1000-2000元=8分，2000-3000元=5分，>3000元=2分 |
| 配件价格 | 与同级车对比评估 |
| 工时费 | 品牌统一标准vs4S店差异化评估 |

#### 1.4 保值率表现（权重：4%）
| 评估指标 | 评分标准 |
|---------|---------|
| 1年保值率 | >85%=10分，80-85%=8分，75-80%=6分，70-75%=4分，<70%=2分 |
| 3年保值率 | >70%=10分，65-70%=8分，60-65%=6分，55-60%=4分，<55%=2分 |
| 5年保值率 | >55%=10分，50-55%=8分，45-50%=6分，40-45%=4分，<40%=2分 |
| 品牌保值口碑 | 历史数据和市场反馈综合评估 |

## 三、评分计算方法

### 3.1 综合得分计算公式

```
综合得分 = Σ(维度得分 × 维度权重)

其中：
维度得分 = Σ(评估项得分 × 评估项权重)
```

### 3.2 得分等级划分

| 综合得分 | 等级 | 评价 |
|---------|------|------|
| 9.0-10.0 | A+ | 极度推荐 |
| 8.5-9.0 | A | 强烈推荐 |
| 8.0-8.5 | B+ | 推荐 |
| 7.0-8.0 | B | 基本满意 |
| 6.0-7.0 | C | 一般 |
| <6.0 | D | 不推荐 |

### 3.3 维度雷达图展示

建议使用雷达图展示6大维度的得分分布，直观对比各车型优劣势。

## 四、评估流程

### 4.1 数据收集阶段
1. 官方参数查询（官网、配置表）
2. 第三方测评报告（汽车之家、懂车帝、易车等）
3. 车主口碑数据（真实用车反馈）
4. 专业媒体试驾测评
5. 市场调研数据（价格、优惠、保值率）

### 4.2 评分阶段
1. 根据评估标准逐项打分
2. 计算各维度得分
3. 计算综合得分
4. 生成对比表格和雷达图

### 4.3 分析阶段
1. 识别各车型优劣势
2. 分析与用户需求的匹配度
3. 提供购买建议

## 五、使用建议

### 5.1 权重调整
根据不同用户群体，可调整各维度权重：

| 用户类型 | 经济性 | 性能 | 空间 | 配置 | 品牌 | 安全 |
|---------|-------|------|------|------|------|------|
| 家庭用户 | 30% | 15% | 25% | 15% | 5% | 10% |
| 年轻群体 | 25% | 25% | 15% | 20% | 10% | 5% |
| 商务用户 | 20% | 20% | 20% | 20% | 15% | 5% |
| 预算敏感 | 35% | 15% | 20% | 10% | 10% | 10% |
| 性能爱好者 | 15% | 35% | 15% | 20% | 10% | 5% |

### 5.2 动态调整
- 新能源车可增加"三电系统"专项评估
- 豪华车可提高"品牌力"权重至20%
- SUV车型可增加"越野能力"专项评估

## 六、附录

### 6.1 数据来源推荐
- 官方渠道：汽车品牌官网、官方APP
- 垂直媒体：汽车之家、懂车帝、易车网、太平洋汽车
- 权威机构：C-NCAP、CIASI、J.D. Power
- 用户口碑：各平台车主口碑区、论坛、社群

### 6.2 注意事项
1. 数据时效性：使用最新数据（6个月内）
2. 数据真实性：交叉验证多个来源
3. 主观因素：专业测评和用户口碑需综合考量
4. 地区差异：考虑不同地区的市场情况和优惠政策
5. 配置版本：明确对比的配置版本（低配/中配/高配）
