# Goal-Driven Agent 用户指南

## 作为用户，你可以做什么？

这个系统就像一个**智能助手**，它会主动帮你完成目标，而不是等你一步步指挥。

## 快速开始

### 1. 告诉系统你的目标

```
用户: 我想在3个月内通过雅思考试，目标是7分
```

系统会自动：
- 创建目标
- 分析需要做什么（维度探索）
- 生成任务列表
- 开始执行

### 2. 系统会主动问你问题

为了制定更好的计划，系统可能会问：

```
🤖 为了帮你制定雅思备考计划，我想了解一下：
   你现在的英语水平怎么样？（比如四六级分数）
```

你回复后，系统会继续：

```
🤖 了解了！那你每天大概能抽出多少时间学习呢？
```

### 3. 系统自动执行任务

根据你的回答，系统会自动：

| 时间 | 系统行为 |
|------|----------|
| 立即 | 搜索最适合你的备考资料 |
| 每天 | 提醒你完成当日学习任务 |
| 每周 | 总结学习进度 |
| 考前1周 | 提醒报名和准备事项 |

### 4. 你会收到通知

当系统完成任务或需要确认时：

```
📬 新通知
   标题: 找到5本推荐的雅思备考书
   内容: 根据你的四级水平，我筛选了这些书籍...

📬 新通知
   标题: 目标可能已完成？
   内容: 你的雅思备考目标已完成80%，要标记为完成吗？
   [标记完成] [继续学习]
```

  1. [进行中] 准备雅思考试 - 50%

  探索维度:

    └─ 考试时间 [待开始]

  任务列表:
    ✓ 搜索备考资料

### 暂停/恢复目标

```
用户: /goal pause 1
系统: 目标"准备雅思考试"已暂停，相关任务将停止执行

用户: /goal resume 1
系统: 目标已恢复，任务调度继续
```

### 回复交互式任务

当系统问你问题时：

```
系统: 你打算什么时候参加雅思考试？
用户: /goal answer 我计划3个月后考，大概6月15日

系统: 收到！我会根据这个时间制定计划。
```

### 标记目标完成

```
系统: 你的雅思目标已完成80%，要标记为完成吗？
用户: /goal complete 1

或者：
用户: 还没，我还想再练习口语
系统: 了解了，我会添加更多口语练习任务。
```

## 不同类型的任务

### 🔍 探索型任务
系统自动搜索信息
- 例："搜索最佳雅思备考书籍"
- 你什么都不用做，等结果通知

### ⏰ 周期性任务
按设定时间自动执行
- 例："每天早上9点提醒学习"
- 系统会持续执行直到目标完成

### 💬 交互型任务
需要你提供信息
- 例：询问你的英语水平、时间安排
- 系统会根据你的回答调整计划

### 👀 监控型任务
持续监控某件事情
- 例："监控雅思报名开放时间"
- 有变化时立即通知你

## 通知管理

### 通知类型

| 图标 | 类型 | 说明 |
|------|------|------|
| 📊 | 报告 | 任务完成的总结报告 |
| ❓ | 求助 | 需要你提供信息 |
| ✅ | 确认 | 需要你确认某件事情 |
| ℹ️ | 信息 | 一般性信息 |
| 🚨 | 紧急 | 需要立即关注 |

### 通知优先级

- **紧急**: 立即显示（如报名截止提醒）
- **高**: 1分钟内显示（如任务完成）
- **中**: 5分钟空闲后显示
- **低**: 用户完全空闲时显示

### 免打扰设置

```
用户: /goal settings

系统:
   当前设置:
   - 免打扰时间: 22:00 - 08:00
   - 通知频率: 实时
   - 最大并发任务: 3

你可以修改:
   - 设置免打扰: /goal settings quiet-hours 22:00-08:00
   - 每日汇总: /goal settings digest daily
   - 每周汇总: /goal settings digest weekly
```

## 常见问题

### Q: 系统会打扰我吗？
A: 不会。系统会检测你是否空闲，重要任务才立即通知，普通任务会等你空闲时推送。

### Q: 我可以同时有多个目标吗？
A: 可以。系统会自动平衡多个目标的执行，优先处理紧急和高优先级的目标。

### Q: 系统怎么知道我完成了某个标准？
A: 系统会自动评估，但会询问你确认。比如达到80%完成度时会问你是否标记完成。

### Q: 如果不满意系统的计划怎么办？
A: 你可以随时反馈。比如："这个计划太紧了"，系统会自动调整。

### Q: 数据存在哪里？
A: 默认存在 `~/.pi/agent/goal-driven/`，完全本地存储，隐私安全。

| `/goal tree <id>` | 查看目标详情树 |
| `/goal pause <id>` | 暂停目标 |
| `/goal resume <id>` | 恢复目标 |
| `/goal stop <id>` | 停止目标 |

| `/goal complete <id>` | 标记目标完成 |
| `/goal settings` | 查看/修改设置 |

## 最佳实践

1. **目标要具体**: "3个月内雅思7分" 比 "学英语" 更好
2. **及时回复问题**: 交互式问题的回复帮助系统制定更好计划
3. **定期检查进度**: 每周看看进度报告，调整计划
4. **反馈很重要**: 告诉系统计划是否合适，它会学习调整
